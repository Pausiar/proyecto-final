# BOOKING CREATOR

- Sistema de creación de sistema de reservas para empresa
- La propia empresa elige que datos poner / imagenes / descipción ponen para su empresa
- Verificación de negocios mediante información no peligrosa del negocio que verifique que este es el negocio real y no un imitador
- Sistema de ranking y reseñas (clasificación cantidad y calidad)
- Sistemas de pago (con intermediaro externo) para mayor personalización y escalar en el ranking de manera fácil
- Aplicación gratuita de base
- Apartado de mis reservas para ver las reservas realizadas
